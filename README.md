### Build process

> [!IMPORTANT]
> Make sure to be in the website's _root directory_.

1. Install [NodeJS](https://nodejs.org/en)
2. Install the local packages:
   
   ```sh
   npm install .
   ```

3. For _Windows_ run:
   
   ```sh
   npm run build
   ```
