= JOST Learn! LHA 2024

https://jostlearn.netlify.app/[Website] for SDG 4 2024 STEAM Project LHA

== Build process

NOTE: Make sure to be in the website's [.underline]#root directory.#

[arabic]
. Install https://nodejs.org/en[NodeJS]
. Install the local packages:
+
[source,sh]
----
npm install .
----
. For *Windows* run:
+
[source,sh]
----
npm run build
----
